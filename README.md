# 👕 T-Shirt Store Project | UCAM Web Programming

Welcome to the **T-Shirt Store** project! This repository contains the source code for the final practical assignment of the Web Programming course at the **Catholic University of Murcia (UCAM)**.

This project aims to build a fully functional e-commerce platform for selling custom t-shirts, managing everything from the customer's shopping experience to the back-office inventory control.

---

## 🚀 Project Overview

The objective of this project is to develop a web application that manages a real business process. Our chosen theme is a **T-Shirt Store**, where we implement a complete cycle of product browsing, user authentication, stock management, and sales reporting.

### Key Features
- **User Authentication:** Registration and login for customers.
- **Inventory Management:** Real-time stock control for different t-shirt designs and sizes.
- **Sales Flow:** Complete shopping cart and checkout process.
- **Reporting:** Automatic generation of invoices and sales reports.

---

## 🛠 Tech Stack & Requirements

The project is divided into two main developmental phases:

### Phase 1: Frontend & Design (HTML5 & CSS3)
* **Structure:** Semantic HTML5.
* **Styling:** Custom CSS3 layout.
* **Canvas API:** A custom-coded brand logo.
* **Interactivity:** Navigation menus, a search bar, and a functional calendar.
* **HTML5 APIs:** Implementation of **Geolocation** (to find nearby pickup points) or **Web Storage** (for persistent shopping carts).

### Phase 2: Backend & Logic (PHP & JavaScript)
* **Server-side:** PHP for business logic.
* **Client-side:** JavaScript and jQuery for dynamic UI updates.
* **Database:** Relational database (MySQL) managed via XAMPP.
* **Security:** Session management for secure user areas.
